# XML Calabash 3

This repository contains the “3.x” version of XML Calabash. This version of XML Calabash
implements [XProc 3.0](https://xproc.org/).

XML Calabash 3.x depends on [Saxon](http://saxonica.com/) 10.x
and some recent version of [JAFPL](http://jafpl.com/).

Welcome to the bleeding edge.
